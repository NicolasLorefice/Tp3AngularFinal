Ingreso de Datos
El número de documento es obligatorio.
El número de documento debe ser menor o igual a 99999999.
El número de documento debe ser mayor o igual a 1000000.
El nombre es obligatorio.
El nombre solo puede contener letras y espacios.
El apellido es obligatorio.
El apellido solo puede contener letras y espacios.
El correo electrónico es obligatorio.
El formato del correo electrónico no es válido.
La fecha de nacimiento es obligatoria.
La fecha de nacimiento debe tener el formato YYYY-MM-DD.
La fecha de ingreso es obligatoria.
La fecha de ingreso debe tener el formato YYYY-MM-DD.
